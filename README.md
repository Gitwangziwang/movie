## 基于Flask框架对微电影网站的开发  

### 项目简述

本项目实现微电影网站的搭建，实现注册、登录、后台管理、电影页面展示等功能。 
采用Python语言编写，Flask框架搭建，MySQL用于存储后台数据
 
### 项目说明

##### 开发环境
* ubuntu (16.04)
* python (3.6.5)
* Flask (1.0.2)
* MySQL (5.7)
##### 数据库设计(models)
* 会员
User
* 电影
Movie
* 上映预告
Preview
* 评论、收藏
Comment、Moviecol

##### 功能设计(views)
1. 后台管理页面（admin）
* 登录（session机制、Form表单验证）
* 修改密码
* 标签管理
* 电影管理
* 预告管理
* 会员管理
* 收藏、评论管理
2. 电影主页面(home)
* 会员注册、登录功能
* 个人资料修改
* 电影信息展示
* 电影视频播放页面
* 收藏、评论功能
* 分页、排序功能

３. 全局功能
* 全局404页面的配置
